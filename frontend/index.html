Azure Resume Architecture

I developed this project as part of the A Cloud Guru Azure Resume Challenge. I utilized Visual Studio Code with the Azure Tools extension pack and built it using .NET 8.0 LTS with an in-process model and the necessary NuGet packages. The core of the solution is hosted in Azure Blob Storage, which contains the static website for the resume. The website is built with HTML, CSS, and JavaScript, with JavaScript implementing a visitor counter. This counter not only displays visitor information but also makes API calls and integrates with an Azure Function. The Azure Function interacts with Azure CosmosDB to store visitor counter data. Additionally, Azure CDN is used to enable features like HTTPS and custom domain support for the static site deployed on Azure Blob Storage. GitHub handles all CI/CD tooling, and the backend is developed in C#.
